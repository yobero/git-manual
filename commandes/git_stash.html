la commande git stash

La commande git stash permet de stocker le dossier et l'index tout en nettoyant le dossier de travail. Elle sauvegarde les modifications locales tout en permettant de revenir au commit de base.

Syntaxe

- git stash

Utlisation

La commande git stash permet de stocker le dossier et l'index tout en nettoyant le dossier de travail. Elle sauvegarde les modifications locales tout en permettant de revenir au commit de base.

- git stash apply

La commande "git stash apply [--index] [-q|--quiet] []" sert à récupérer le dossier stocké dans "stash" et à le ramener à l'étape de travail présente. Si rien n'est tapé à la place de "stash", récupérera le dernier dossier stocké .

Exemple de l'affichage de la commande dnas le terminal

- git stash save

git stash save est l'équivalent de d'un git stash auquel on n'aurait pas précisé où classer la sauvegarde . Cette commande va stocker le travail en cours dans un dossier nommé "WIP on nom_de_la_branche". Cependant, ajouter à la fin de la ligne " []" changera le nom de l'emplacement de stockage en "message" .

Affichage

Exemple de l'affichage de la commande dnas le terminal

- git stash list

liste les stashs que l'on a actuellement fait.chaque stash est listé et nommé.par exemple stash@{0} est le dernier stash, et stash@{1} est celui avant. il y a le nom de la branche utilisé quand le stash a été fait et une courte description.

Affichage

stash@{0}: WIP on submit: 6ebd0e2... Update git-stash documentation
stash@{1}: On master: 9cc0589... Add git-stash

- git stash drop

abandonne le stash selectioné.

Affichage

$ git stash list stash@{0}: WIP on master: 049d078 added the index file stash@{1}: WIP on master: c264051... Revert "added file_size" stash@{2}: WIP on master: 21d80a5... added number to log $ git stash drop stash@{0} Dropped stash@{0} (364e91f3f268f0900bc3ee613f9f733e82aaed43)
