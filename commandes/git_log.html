Git log

Git log vous donne la liste des commits qui ont été effectués dans le répertoire. Cela vous permettra de voir les modifications qui ont été faites. Cela vous donnera aussi le numéro de commit qui pourra vous servir pour d'autres commandes.

Syntaxe

git log
