git fetch

Cette commande importe des commits à partir d'un répo à distance dans votre répo local. Les commits résultants sont stockés sous forme de branches à distance à la place des branches locales avec lesquelles vous avez travaillé.

Syntaxe

git fetch

git fetch

Affichage

[remote "origin"] fetch = +refs/heads/*:refs/remotes/origin/*

git fetch -v

git fetch --all

Cette commande permet de "fetch" toutes les branches à distance.

Syntaxe

git fetch --all

git fetch --all

Affichage

Fetching origin From ssh://gerrit.wikimedia.org:29418/integration/jenkins c75cd9d..4956b35 master -> origin/master Fetching gerrit From ssh://gerrit.wikimedia.org:29418/integration/jenkins c75cd9d..4956b35 master -> origin/master

git fetch --dry-run

Cette commande vous montre le résultat d'un "fetch" en ne modifiant rien, elle est utilisée simplement pour simuler une action.

Syntaxe

git fetch --dry-run

git fetch --dry-run

Affichage

[remote "origin"] fetch = +refs/heads/*:refs/remotes/origin/*

git fetch -r
