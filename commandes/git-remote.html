git remote

  • Accueil

La commande permet d'afficher la ou les branches qui vous suivent lorsque vous commitez

Syntaxe

git remote

Affichage

git remote
origin

git remote add

La commande permet d'ajouter un depot à un nomCourt pour faciliter son utilisation Ensuite, si vous voulez recuperer des information sur un URL et que vous ayez utilise git remote add vous pouvez utiliser le nomCourt lié à cette URL en utiilisant la commande git fetch nomCourt

Syntaxe

git remote add
git remote add [nom] [URL]
nom URL

Autre exemple
git remote add test https://github.com/nomUtilisateur/projetX
test https://github.com/nomUtilisateur/projetX

git remote rename

Cette commande permet de renomer de nomCourt pour à autre nom. On va reprendre l'exemple précédant. Vous pouvez toujours supprimer la réference avec la commmande git remote rm nomCourt

Syntaxe

git remote rename ancienNomCourt nouceauNomCourt
git remote add test https://github.com/nomUtilisateur/projetX
test https://github.com/nomUtilisateur/projetX
git remote
origin
test

git remote rename test pjtX
git remote
origin
pjtX

git remote remove

Git remote remove est utilisé afin de supprimer ce que le remote lui même a fait. Vous avez fait un remote et donc donné un nom à ce remote. Si vous voulez supprimer ce que vous venez de faire avec le remote, il faut juste taper git remote remove dans la console afin de supprimer cela.

Syntaxe

git remote remove (et le nom du remote)
$ git remote remove
def**
origin
upstream
$ git remote remove upstream
$ git remote
defeo
origin

git remote set-url

Git remote set-url change l'url pour le remote. Si aucune url n'est déjà présente, la fonction crée une nouvelle url. Par contre si il y a déjà une url et on utilise cette fonction en ne mettant pas le bon nom, alors la console vous affichera une erreur et aucun changement ne sera effectué.

Syntaxe

git remote set-url
$ git remote set-url
usage: git remote set-url[--push] []
or: git remote set-url --add
or: git remote set-url--delete

--push manipulate push URLs
--add add URL
--delete delete URLs
